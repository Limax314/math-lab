# Math Lab 🧮

`Lean`을 활용한 수학 문제 풀이 연구 및 저장소입니다.

## 🎯 목표

핸드폰 갤러리에 쌓여만 가는 수학 문제들을 방치하지 않고, 최신 도구를 활용해 디지털화하여 풀이 과정을 기록하고 관리합니다.

## 🛠️ 도구

- **Lean 4**: 수학적 정리와 증명을 엄밀하게 형식화합니다.
- **LaTeX**: 수식과 풀이 과정을 깔끔하게 조판합니다.

## 📂 폴더 구조

- `problems/*/*.lean`: 수학 문제 풀이 과정(`.lean` 파일)
- `problems/*/*.tex`: 조판된 풀이 문서 (`.tex` 파일)
